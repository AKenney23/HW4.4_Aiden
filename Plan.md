# Assignment and Repo Plane

This plan has two parts: Part 1 covers the goals and plans for the completion 
of HW #4.3 and Part 2 covers the goals and plans for setting up and organizing
this GitHub repository for HW #4.4.

## Part 1: HW #4.3 Goals and Plan

### Goal: Writing a QMD file that renders to a PDF report containing analysis about three situations:
1. Busiest Airports from 2020 to 2025
2. Monte Carlo Integration
3. Comparison of GenAI responses for a given data set.

### Needs:

#### Nouns: 
- Data from Wikipedia about busiest airports from 2020 to 2025
- `calcium.csv` used for comparison of GenAI prompts
- Images comparing the GenAI Responses
- Rendered PDF output
- R Packages: `tidyverse`, `ggplot2`, `knitr`, `kableExtra`, `rvest`
- Alt text

#### Verbs: 
- Web scraping using `rvest`
- Data manipulation using `tidyverse`
- Data visualization using `ggplot2`
- Table formatting using `kableExtra`
- Running Monte Carlo simulations with `runif`

### Steps:
1. Web scrape data about busiest airports from Wikipedia using `rvest`
2. Clean and manipulate the data using `tidyverse`.
3. Create visualizations of the busiest airports data using `ggplot2`.
4. Run Monte Carlo simulations using `runif` on different resolutions
